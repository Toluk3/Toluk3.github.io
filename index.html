Turnera medica

La Turnera Médica es una aplicación diseñada basada en Java para gestionar turnos médicos y pacientes, enfocándose en: la programación orientada a objetos, la arquitectura de las 3 capas y el uso de interfaces gráficas con Swing. Incorpora una base de datos H2 para el manejo local de datos y gestión de sesiones de usuarios.

  • -> Lenguaje usado Java
  • -> Programación orientada a objetos
  • -> Arquitectura de las 3 capas (DAO, Services, GUI)
  • -> Uso de interfaces gráficas con Swing
  • -> Incorpora una base de datos H2
